## 🛠 Jak użyć

1. Zaimportuj potrzebny model:
```python

```

2. Stwórz i wytrenuj model:
```python
model = DecisionTreeRegressor(max_depth=3)
model.fit(X_train, y_train)
```

3. Dokonaj predykcji:
```python
predictions = model.predict(X_test)
```
